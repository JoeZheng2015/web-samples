# 绝对定位基于边缘

项目[示例](https://zhengjunxin.github.io/web-samples/src/position_absolute/) ([源文件](https://github.com/zhengjunxin/web-samples/blob/master/src/position_absolute/index.html))

绝对定位元素是基于 relative 父元素的边缘，即父元素有 padding, border, margin 时，left 和 top 是基于 padding 的

理解：padding 属于元素内，border 和 marin 属于元素外了
